# Linuxmint-Panel-Top-Position
* Change panel to top Position

## How To

###### Download Package
* Open Terminal
* $ cd Linuxmint-Panel-Top-Position
* $ ./install

###### Git Clone
* Open terminal
* $ git clone https://github.com/Litle-Panda/Linuxmint-Panel-Top-Position.git
* $ cd Linuxmint-Panel-Top-Position
* $ ./install
